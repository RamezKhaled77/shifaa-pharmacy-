
# Shifaa Pharmacy

A vanilla JavaScript E-commerce application for a pharmacy.

## Structure

- **index.html**: Main entry point.
- **css/**: Stylesheets using a modular architecture.
- **js/**: Application logic.
  - **views/**: Page components.
  - **components/**: Reusable UI parts.
  - **services/**: Business logic (Cart, Storage).
  - **data/**: Mock data.

## Running

1. Open `index.html` in a modern web browser.
2. For better routing support, serve with a local server (e.g., Live Server).
